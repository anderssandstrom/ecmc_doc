Adds an Ether\-C\-A\-T slave to the hardware configuration.\par
. 

Each added slave will be assigned an additional index which will be zero for the first successfully added slave and then incremented for each successful call to \char`\"{}\-Cfg.\-Ec\-Add\-Slave()\char`\"{}.\par
